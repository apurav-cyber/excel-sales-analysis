# Excel Sales Analysis Dashboard

## Project Overview

This project analyzes sales data using Microsoft Excel.

## Skills Used

- Pivot Tables
- Pivot Charts
- Slicers

## Objectives

- Analyze sales performance
- Identify top-selling products
- Compare monthly sales
- Visualize key business metrics
